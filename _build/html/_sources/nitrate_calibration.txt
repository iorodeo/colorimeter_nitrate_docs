Nitrate standard dilutions
=================================

Below are methods for preparing 5 mL or 10 mL nitrate standards in three different concentration ranges. Adjust volumes as necessary for preparing larger volumes. 


#. Low-range:  0 - 40 ppm e.g. Hach NitraVer5, API test kit, Nitrate reductase method
#. Mid-range:  0 - 80 ppm e.g. Hach NitraVer5, API test kit.
#. High-range: 0 - 200 ppm e.g. API Test Kit


In each case, the first step is preparing a working stock by diluting a commercially available nitrate standard. This working stock is then diluted further resulting in a set of 8 nitrate concentrations (plus a distilled water blank).
In addition to the equipment listed on the appropriate nitrate assay page, you will also need the following equipment and reagents if you plan to collect nitrate calibration data:

* 1,000 ppm N (4,430 ppm as NO\ :sub:`3`\ ) standard, Labchem Cat # LC18570-1
* 100 mL volumetric flask for preparing the working stock
* 5 mL or 10 mL sample tubes
* Distilled water

 

Low-range (0.00 - 40 ppm NO\ :sub:`3`\)   
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

* Volume: 5 mL
* 40 ppm NO\ :sub:`3`\  working stock: 1.129 mL of the 4,430 ppm (1,000 ppm as N) standard in 100 mL distilled water in volumetric flask. Fill to the line with distilled water.
